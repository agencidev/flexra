# Flexra - AI och Automation

Ett Next.js-projekt byggt med Relume UI-komponenter.

## Komma igång

Först, installera beroenden med pnpm:

```bash
pnpm install
```

Kör sedan utvecklingsservern:

```bash
pnpm dev
```

Öppna [http://localhost:3000](http://localhost:3000) i din webbläsare för att se resultatet.

## Teknisk stack

- **Next.js 14** - React-ramverk med App Router
- **Relume UI** - Komponentbibliotek
- **Tailwind CSS** - Styling med Relume preset
- **JavaScript** - Ingen TypeScript
- **pnpm** - Pakethanterare

## Projektstruktur

```
flexra/
├── app/
│   ├── layout.js      # Root layout
│   ├── page.js        # Hemsida
│   └── globals.css    # Globala stilar
├── components/        # Alla Relume-komponenter
├── package.json
├── tailwind.config.js
└── next.config.js
```
